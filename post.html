HTML | CSS | JavaScript | React

Bir web sitesi belirli bölümlerden oluşmaktadır. Her bölümün oluşturulmasında kullanılan özel diller mevcuttur. HTML bir veri düzenleme prosedürüdür denilebilir. Çünkü programlama dilleri bir seri prosedürden oluşur ancak HTML verinin kendisini oluşturmaktadır. HTML web sayfalarının içeriklerini yüklemeyi sağlamaktadır. Bu da web sitesinin iskeletini oluşturur. İçeriklerin ve resimlerin nasıl yerleşeceğini, hangi stilde gösterileceğini yazı fontu gibi verileri belirtir. HTML etiketlerden oluşur “<” ve “>” karakterleri arasında etiketler yer alır. Açma etiketleri “<” ve “>” karakterleri arasına yazılırken kapatma etikerleri “” karakterleri arasına yazılır. gibi tek kullanılan etiketler olduğu gibi ve gibi etiketlerde bulunmaktadır. Açma kapama özelliğindeki etiketler arasında yazılan değerlere etki eder.

CSS dili web sitelerinde özellikle katmanlı stillerin oluşturulmasında kullanılmaktadır. Güçlü ve kullanıcı dostu bir dil olması nedeniyle tercih edilmektedir. Kullanıcı dostu olmasının yanı sıra html etiketleri ile yapılamayan değişikliklerin yapılmasına imkan sağlamaktadır. Peki nedir bunlar örnek vermek gerekirse arkaplan resmi ekleme bunlardan birisidir. Web sitelerinin herhangi bir sayfasının nasıl konumlanacağı, arka plan resimleri ve renkleri, kenarlıkların konumları ve renkleri CSS yardımı ile ayarlanmaktadır. Kullanıcı dostu olması sayesinde topluca ya da tek tek özellikleri kontrol etme imkanı sağlamaktadır. CSS ile önceden yazılmış olan HTML etiketlerini etkilemek mümkündür. İçeriklerin özellikleri ile ilgili değişiklik yapmak ve HTML değerlerini baskılamak mümkündür.

JavaScript web sitelerinin davranışlarını yönetmekte kullanılmaktadır. Statik ve durağan siteler hazırlanmasına neden olan HTML bir süre sonra yeterli gelmemektedir. Kullanıcıya geri bildirim göndermek, kullanıcının hareketlerine göre sitede değişiklikler yapmak gibi çeşitli işlemleri gerçekleştirebilmek için JavaScript kullanılmaktadır. Kendine ait .Js uzantılı dosya ile koda dahil edilebildiği gibi “” etiketleri arasına yazılarak da HTML koduna dahil edilebilmektedir. JavaScript her tarayıcıda aynı performansta çalışmayabilmektedir. Web sitelerinin altında gördüğünü “en iyi X tarayıcısında çalışır” etiketleri bu yüzden yer almaktadır. Ancak son zamanlarda bu sorunlar çözülerek daha güçlü web sitelerinin her tarayıcıda etkili bir şekilde çalışması sağlanmaktadır. Kaliteli bir tasarım için HTML-CSS-JavaScript güçlü özellikleri beraber kullanılmalıdır.

React açık kaynak olan, kullanıcı arayüzleri yaratmak için kullanılan bir Javascript kütüphanesidir. Arayüz yaratmak adına yeniden kullanılabilen bileşenler, component adı verilen yapılar kullanılmaktadır.
